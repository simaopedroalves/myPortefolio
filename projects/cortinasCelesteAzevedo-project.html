CORTINAS CELESTE AZEVEDO

Welcome to the website of Celeste Azevedo, a talented seamstress who specializes in creating beautiful curtains and other textile products.

This website was developed to help Celeste online bussiness and for me, to practice and apply React Hooks and TailwindCss.

It was a great experience because I deepened my knowledge of useState(), useEffect() and useRef().

This project also allowed me to reuse components and build an efficient website.

Another key point was styling with TailwindCss. I'm a fan now! The easy way with witch this framework allows to write CSS directly in HTML is amazing.

During the development, I struggled with structuring the differents files and play with TailwindCss configurations to understand how this specific framework could be used.

In addition, the website is responsive and adapts to different screen sizes, ensuring a good experience for users on all devices.
